Meu aprendizado na Trybe

Meu aprendizado na trybe vem crescendo de forma significativa. Meu conhecimento em front-end era quase zero e após poucas semanas eu estou conseguindo criar páginas usando HTML e CSS, fazendo o versionamento do código com o Git e GitHub e utilizar o Unix.

Além das hard-skills a trybe trabalha com com Soft-skills, onde já pude me atentar para vários pontos de melhoria que preciso exercitar para poder me ser bem avaliado em um processo seletivo, ou mesmo dentro da empresa.

Uma passsagem na trybe
